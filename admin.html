Email Integration Setup
Чтобы входящие письма от пользователей появлялись здесь автоматически, настрой следующее:
1. Cloudflare Email Routing
dash.cloudflare.com → Email Routing → создай адрес support@bb-vpn.com и направь на Cloudflare Worker
2. Worker обработчик
Worker получает письмо, парсит его и сохраняет в D1 / KV — откуда эта вкладка читает тикеты
3. Resend для ответов
Кнопка Send вызывает POST /api/support-reply → Worker отправляет через Resend от имени support@bb-vpn.com
